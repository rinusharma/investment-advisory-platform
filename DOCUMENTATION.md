#Investment Advisory Platform — Documentation

## 1. How to Run the Application

### Prerequisites
- Node.js (v16+ recommended)
- npm


### Backend Setup

```bash
cd server
npm install
npm run dev

Runs backend on:

http://localhost:5000

### Frontend Setup
cd client
npm install
npm run dev

Runs frontend on:

http://localhost:5173

Application Flow
Open frontend in browser
Upload client portfolio or add notes
Complete risk assessment chat
Run AI simulation pipeline
View scoring & recommendations
Analyze dashboard insights
